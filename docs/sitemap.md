# Sitemap — databerganalytics

> The URL structure. Kept deliberately shallow. Every URL serves a single job.

## Languages

- English: default at the root (`/`).
- Dutch: prefixed at `/nl/`.
- Language toggle in `Nav` and `Footer` preserves the current route and swaps the locale prefix.
- `<html lang>` set per request. `hreflang` alternates on every page.

## Routes

| EN                         | NL                            | Job                                                  |
|----------------------------|-------------------------------|------------------------------------------------------|
| `/`                        | `/nl/`                        | Home — hero, signature dashboard, services, proof    |
| `/services`                | `/nl/diensten`                | Service overview (all services listed)               |
| `/services/[slug]`         | `/nl/diensten/[slug]`         | Single service detail                                |
| `/work`                    | `/nl/cases`                   | Case-study index                                     |
| `/work/[slug]`             | `/nl/cases/[slug]`            | Single case study                                    |
| `/about`                   | `/nl/over-ons`                | Team, principles, how we work                        |
| `/contact`                 | `/nl/contact`                 | Contact form + book-a-call link                      |
| `/insights` *(Phase 11)*   | `/nl/insights` *(Phase 11)*   | Blog index — only if ≥3 real posts                   |
| `/insights/[slug]`         | `/nl/insights/[slug]`         | Blog post                                            |
| `/404`                     | `/nl/404`                     | Warm, not cute. Route back to home + contact.        |

## URL conventions

- All lowercase. Hyphenated slugs. No trailing slashes (Astro default).
- Slugs are stable — if a case study's client name changes, we do not rename the slug.
- No parameters in canonical URLs. Filters on `/work` use query strings and are marked `noindex` when filters are active.

## Sitemap.xml

- Generated by `@astrojs/sitemap` in Phase 14.
- Excludes `/_dev/*` and any draft content.
- Both language variants listed with `<xhtml:link rel="alternate" hreflang>`.

## robots.txt

- Allow all. Point to `/sitemap-index.xml`.
- Disallow `/_dev/` and any staging paths.
